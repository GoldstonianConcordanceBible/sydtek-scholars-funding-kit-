# Vendor Onboarding

## Minimum requirements
- named account owner + support contact
- data handling terms (privacy/security)
- uptime expectations
- billing terms
- termination/offboarding plan

## Risk checks
- Does vendor store student data? If yes, restricted handling rules apply.
- Are exports/backups available?
- Can we revoke access quickly?

## Approval
- Program Director + Security Lead approval required for any vendor that touches restricted data.
