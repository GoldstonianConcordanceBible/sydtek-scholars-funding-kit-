# Information Security Policy (Baseline)

## Purpose
Protect student, family, and staff information.

## Access control
- Least privilege access to systems and folders
- Separate accounts per staff member (no shared logins)
- Remove access within 24 hours of role end

## Passwords & MFA
- Use strong passwords and MFA wherever available
- Do not store credentials in GitHub
- Use a password manager: [TBD]

## Devices
- Keep OS updated
- Use device lock and encryption when possible
- Do not store restricted data on unmanaged devices

## Data sharing
- Share restricted files only through approved secure channels
- No posting student data in chat tools
